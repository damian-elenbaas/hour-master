About

Hour Master is een applicatie dat gebouwd is voor bedrijven in de bouw. Deze applicatie biedt de mogelijkheid voor bedrijven om urenregistratie makkelijker te maken. Werknemers die aan projecten werken kunnen hun uren aangeven die ze aan een project hebben gewerkt. Daarbij kunnen ze aangeven welke werkzaamheden ze hebben uitgevoerd en of er eventueel machines zijn gebruikt. Dit geeft een beter inzicht voor het kantoor om een kostenplaatje te maken voor de klant en eventueel vroegtijdig onderhoud in te plannen voor de machines.

Uitleg

We hebben in dit project te maken met verschillende entiteiten. Die zijn als volgt:

Gebruiker

De gebruiker is het hart van de applicatie. Een gebruiker kan een kantoormedewerker of een stratenmaker zijn. Deze heeft een naam, email en wachtwoord. Daarnaast heeft een gebruiker een rol. Deze rol bepaald welke type gebruiker ze zijn (Kantoormedewerker of stratenmaker).

Project

Een project is een locatie waar stratenmakers aan het werk zijn. Een project heeft een naam en een locatie. Daarnaast heeft een project een kantoormedewerker die verantwoordelijk is voor het project.

Locatie

Een locatie is een plek waar een project zich bevindt. Een locatie heeft een adres, postcode en stad.

Urenschema

Een urenschema is een schema waarin een stratenmaker zijn uren kan registreren. Een urenschema heeft een datum en een project. Daarnaast heeft een urenschema een stratenmaker die verantwoordelijk is voor het urenschema.

Urenschema Rij

Een urenschema rij is een rij van een urenschema. Dit is een los entiteit, omdat je bij meerdere projecten op een dag werkzaamheden kan hebben uitgevoerd. Bij een urenschema rij wordt er een omschrijving van de werkzaamheden ingevuld met de gemaakte uren. Daarnaast kan er eventueel een machine er aan verbonden worden, als deze gebruikt is op dat moment

Machine

Een machine spreekt voorzich. Deze wordt gebruikt bij werkzaamheden en kan aan een urenschema rij gekoppeld worden. Voor een machine wordt de naam en typenummer bijgehouden. Daarnaast ook het aantal uren dat gedraait moet worden, voordat er onderhoud nodig is. Daarnaast wordt er bijgehouden hoeveel keer er onderhoud is gedaan aan de machine, zodat er berekend kan worden wanneer het volgende moment is.
